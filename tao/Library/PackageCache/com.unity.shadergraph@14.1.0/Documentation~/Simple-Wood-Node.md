Simple Wood 节点
===================

[](#description)描述
---------------------------

基于输入 **UV** 生成简单木纹（Simple Wood）。

[](#ports)端口
---------------

| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| UV | 输入 | Vector 2 | UV | 输入的 UV 值 |
| Factor | 输入 | Vector 4 | 无 | 控制木纹纹理的频率和强度 |
| Distort | 输入 | Float | 无 | 控制木纹的扭曲或变形程度 |
| Out | 输出 | Float | 无 | 输出值 |
