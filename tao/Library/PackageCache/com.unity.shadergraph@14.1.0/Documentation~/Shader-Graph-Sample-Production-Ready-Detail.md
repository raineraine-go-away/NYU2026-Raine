细节
======

在此上下文中，细节指的是添加到地形上的网格，例如草、杂草、灌木、卵石等。要了解更多信息，请阅读关于[细节](https://docs.unity.cn/cn/tuanjiemanual/Manual/terrain-Grass.html)的地形文档。细节网格对着色器有一些特定的要求。首先，由于这些网格在地形中使用的数量较多，我们必须尽可能使它们的着色器快速高效。这主要意味着保持纹理采样的数量较低，并在顶点着色器中进行更多处理，而不是在像素着色器中进行。其次，由于这些网格在特定距离停止渲染并消失，我们必须使用一种方法使其逐渐消失，以防止突然消失的效果，使其不那么明显。在每个着色器中，您将看到使用了距离遮罩或距离截止节点来创建一个在网格停止渲染之前在一定距离内使其逐渐消失的遮罩。

#### 苜蓿

苜蓿（Clover）的着色器只使用一个单通道纹理，以降低成本并节省纹理内存。颜色通过在纹理的灰度值之间线性插值生成明亮和暗色之间的过渡。每个苜蓿实例使用略微不同的颜色变化。我们根据每个实例的位置生成一个随机值，用于为每个实例提供颜色变化。距离遮罩在顶点着色器中计算，以节省性能，然后传递给像素着色器，在那里与纹理和一些屏幕空间噪声结合在一起。这些元素一起传递到淡入淡出过渡节点，使网格在剪切偏移和剪切距离值之间逐渐消失。

#### 蕨类植物

蕨类植物（Ferns）着色器使用颜色、法线和遮罩纹理来定义蕨类植物材质。它根据风的设置来动画蕨类植物。它还创建了次表面散射效果，使蕨类植物的叶片在阳光照射下反面被照亮。对于环境光遮蔽，我们在靠近地面的地方加深了 AO。与其他细节着色器一样，我们在远离时也会使蕨类植物逐渐消失，以防止它们突然出现。

#### 草

通常，草（Grass）是使用草纹理创建的广告牌。这种着色器不同。我们为每片草使用网格。为了尽可能降低网格的成本，我们的草叶网格只有 12 个顶点和 10 个三角形。它们没有 UV 坐标、法线或顶点颜色，因此网格中存储的唯一数据是位置。网格尽可能简单。我们还在顶点着色器中尽可能多地进行处理，以降低成本。风、颜色、透明度和距离渐变都在顶点着色器中计算。

着色器生成风力，然后使用它们来弯曲草叶。风力的方向和阵风强度各不相同，因此草叶的运动感觉自然。

#### 荨麻

荨麻（Nettle）着色器用于简单的宽叶灌木。它是蕨类植物着色器的一种变体，因此具有相似的特性。主要区别在于它已被调整为仅使用一个纹理采样，以减少纹理内存使用和着色器成本。纹理在红色和绿色通道中包含法线 X 和 Y。蓝色通道是不透明度与用于调节光滑度、AO 和颜色的灰度遮罩的组合。

#### 卵石

与其余细节着色器一样，卵石（Pebbles）着色器的设计旨在尽可能降低成本。它只使用一个小的噪声纹理。它使用噪声纹理和实例 ID 创建颜色变化，以便每个卵石簇都有其独特的颜色。并且它会在一定距离处逐渐消失卵石，以防止突然出现。
