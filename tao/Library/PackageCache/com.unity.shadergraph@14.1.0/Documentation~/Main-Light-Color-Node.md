Main Light Color 节点 
================  

**描述**  
----  
获取场景中主光源的颜色信息。输出的颜色信息包含灯光的亮度、色彩以及其他相关属性，用于计算材质的直接光照效果。


**端口**  
------  
| 名称   | 方向  | 类型       | 描述                                |  
|--------|-------|------------|-------------------------------------|  
| Out    | 输出  | Vector 3   | 主光源的颜色信息，包含亮度与色彩属性 |
