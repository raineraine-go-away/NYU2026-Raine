
Bricks 节点  
================  

描述  
----  
生成基于输入 **UV** 的砖块状纹理（Bricks）。您可以通过调整 **Tiling** 参数控制砖块的排列密度，通过 **Size** 参数设置砖块的比例大小，并使用 **Joint X** 和 **Joint Y** 参数调整砖块之间的水平和垂直接缝宽度。**Offset** 参数用于设置砖块的水平偏移，而 **Luminance** 输入可以控制砖块颜色的明暗变化。  

端口  
------  
| 名称        | 方向  | 类型     | 绑定 | 描述                                    |  
|-------------|-------|----------|------|-----------------------------------------|  
| UV          | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于砖块纹理映射       |  
| Tiling      | 输入  | Vector 2 | 无   | 控制砖块的排列密度（X 和 Y）           |  
| Size        | 输入  | Float    | 无   | 控制砖块的大小比例                     |  
| Joint X     | 输入  | Float    | 无   | 控制砖块之间的水平接缝宽度             |  
| Joint Y     | 输入  | Float    | 无   | 控制砖块之间的垂直接缝宽度             |  
| Offset      | 输入  | Float    | 无   | 控制砖块的水平偏移                     |  
| Luminance   | 输入  | Vector 2 | 无   | 控制砖块的颜色明暗变化                 |  
| Out         | 输出  | Float    | 无   | 输出值       |  
| Shelf       | 输出  | Float    | 无   | 输出砖块接缝的附加信息，例如额外纹理   |  

控件  
------  
| 名称        | 类型     | 选项          | 描述                                    |  
|-------------|----------|---------------|-----------------------------------------|  
|  | 下拉菜单 | Shelf、Luminance | 设置砖块纹理的输出模式（接缝或明暗）   |
