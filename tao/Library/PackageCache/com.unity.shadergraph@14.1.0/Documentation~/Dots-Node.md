Dots 节点
================  

描述
----  
生成基于输入 **UV** 的点状图案（Dots）。您可以通过调整 **Tiling** 参数控制点的平铺密度，通过 **Offset X** 和 **Offset Y** 参数实现点的位置偏移，并通过 **Size** 参数控制点的大小。

端口  
------  
| 名称   | 方向  | 类型 | 绑定 | 描述   |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制点状纹理的平铺密度（X 和 Y）       |  
| Offset X     | 输入  | Float    | 无   | 控制点在 X 方向上的偏移                |  
| Offset Y     | 输入  | Float    | 无   | 控制点在 Y 方向上的偏移                |  
| Size         | 输入  | Float    | 无   | 控制点的大小                           |  
| Out          | 输出  | Float    | 无   | 输出值                       |
