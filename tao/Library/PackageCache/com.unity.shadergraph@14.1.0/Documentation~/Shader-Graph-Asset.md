Shader Graph 资源
===============

描述
--

**Shader Graph 资源**是随 Shader Graph 引入的新**资源**类型。可从 [Project 窗口](https://docs.unity.cn/cn/tuanjiemanual/Manual/ProjectView.html)中的 **Create** 菜单创建 **Shader Graph 资源**。


为方便起见，**Blank Shader Graph** 和 [Sub\-graph](Sub-graph.md) 都有 **Create** 菜单项。这些菜单项位于 **Shader** 子菜单中。渲染管线可能会提供其他选项。这些选项将创建一个新的 Shader Graph，其中包含所选着色模型所需的设置和主栈中的 Block 节点。


可通过双击 **Shader Graph 资源**或在选中 **Shader Graph 资源**时单击 [Inspector](https://docs.unity.cn/cn/tuanjiemanual/Manual/UsingTheInspector.html) 中的 **Open Shader Editor** 来打开 [Shader Graph 窗口](Shader-Graph-Window.md)。
