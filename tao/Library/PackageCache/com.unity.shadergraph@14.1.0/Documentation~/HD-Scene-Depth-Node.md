高清场景深度节点
===================

[](#description)描述
---------------------------

高清场景深度节点（HD Scene Depth Node）使用UV输入访问当前摄像机的深度缓冲区。Unity预期该值为标准化的屏幕坐标。您还可以使用此节点访问深度缓冲区中的mipmap。

该节点只能在片元着色器阶段使用，且仅适用于非不透明材质。

[](#render-pipeline-compatibility)渲染管线兼容性
---------------------------------------------------------------

| **节点** | **通用渲染管线 (URP)** | **高清渲染管线 (HDRP)** |
| --- | --- | --- |
| **HD Scene Color** | 否 | 是 |

[](#ports)端口
---------------

| **名称** | **方向** | **类型** | **绑定** | **描述** |
| --- | --- | --- | --- | --- |
| **UV** | 输入 | Vector 4 | 屏幕位置 | 设置要采样的标准化屏幕坐标。 |
| **Lod** | 输入 | Float | 无 | 设置采样深度缓冲区时使用的mip层级。 |
| **Output** | 输出 | Vector 3 | 无 | 输出值。 |

[](#depth-sampling-modes)深度采样模式
---------------------------------------------

| 名称 | 描述 |
| --- | --- |
| **Linear01** | 0到1之间的线性深度值 |
| **Raw** | 原始深度值 |
| **Eye** | 转换为视空间单位的深度值 |

[](#notes)注意
---------------

要在自定义渲染管线中使用高清场景深度节点，必须显式定义其行为，否则它会返回白色。
