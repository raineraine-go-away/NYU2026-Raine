
Arctangent 节点
=============


描述
--


返回输入 **In** 的值的反正切值。每个分量都应在 \-Pi/2 到 Pi/2 的范围内。


端口
--




| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| In | 输入 | 动态矢量 | 输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Arctangent_float4(float4 In, out float4 Out)
{
    Out = atan(In);
}

```
