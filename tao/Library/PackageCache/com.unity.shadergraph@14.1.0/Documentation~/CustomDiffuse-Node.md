CustomDiffuse 节点 
================  

**描述**  
----  
用于计算用户自定义的固有色光照效果。通过输入固有色、灯光颜色、阴影信息、法线和灯光方向，用户可以灵活定义光照的直接漫反射效果，适配不同的自定义材质需求。


**端口**  
------  
| 名称                  | 方向  | 类型       | 绑定          | 描述                                        |  
|-----------------------|-------|------------|---------------|---------------------------------------------|  
| Diffuse              | 输入  | Vector 3   | 无            | 输入固有色信息，通常为 Albedo 数据         |  
| Light Color          | 输入  | Vector 3   | 无            | 输入灯光颜色信息，可以输入颜色信息，也可根据需求生成特定的灯光颜色        |  
| Light Attenuation    | 输入  | Float      | 无            | 输入灯光的阴影信息，如实时灯光，混合灯光以及shadowmask等 |  
| Normal WS            | 输入  | Vector 3   | World Space   | 输入世界空间的法线信息                      |  
| Light Direction WS   | 输入  | Vector 3   | World Space   | 输入世界空间的灯光角度信息                    |  
| Out                  | 输出  | Vector 3   | 无            | 输出自定义的漫反射照明信息                  |
