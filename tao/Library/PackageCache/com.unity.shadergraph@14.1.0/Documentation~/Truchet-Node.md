Truchet 节点 
================  

**描述**  
----  
生成基于输入 **UV** 的 Truchet 图案纹理，方便制作具有布纹的织物。通过调整 **Tiling** 参数控制图案的平铺密度，**Repetition** 参数设置图案的重复次数，**Line Width** 参数控制线条的宽度，**Line Edge** 参数调节线条边缘的平滑程度，**Seed** 参数用于生成不同的随机图案。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制图案的平铺密度（X 和 Y）           |  
| Repetition   | 输入  | Float    | 无   | 控制图案的重复次数                     |  
| Line Width   | 输入  | Float    | 无   | 调整图案线条的宽度                     |  
| Line Edge    | 输入  | Float    | 无   | 控制线条边缘的平滑程度                 |  
| Seed         | 输入  | Float    | 无   | 用于生成不同的随机图案                 |  
| Out          | 输出  | Float    | 无   | 输出值                  |
