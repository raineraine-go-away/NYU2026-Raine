端口
==


描述
--


**端口**定义了[节点](Node.md)（Node）上的输入或输出。将[边](Edge.md)（Edges）连接到端口可让数据流经 Shader Graph 节点网络。


每个端口都有一个[数据类型](Data-Types.md)用于定义可以连接到的边。每种数据类型都有一个相关的颜色用于标识其类型。


任何一个**输入端口**只能连接一条边，但一个**输出端口**可以连接多条边。


通过使用鼠标左键从端口拖动一条边并将其放到工作空间的空白区域中，即可打开 [Create Node Menu](Create-Node-Menu.md) 上下文菜单。


### 默认输入

每个表示用于将数据输入节点的**输入端口**（节点左侧的端口）都具有**默认输入**。在未连接边时，这显示为连接到**端口**的小字段。除非端口具有[端口绑定](Port-Bindings.md)，否则此字段将显示端口[数据类型](Data-Types.md)的输入。如果端口具有[端口绑定](Port-Bindings.md)，则默认输入字段可能会显示一个特殊字段，例如用于选择 UV 通道的下拉表单，或仅显示可帮助您了解预期输入的标签，例如几何数据的坐标空间标签。
