## Shader Stats

Shader Stats 功能用于查看 shader 编译后的详细统计数据。启用右上角的 Stats 功能后，详细数据将显示在工作空间左下角。

> 该功能目前处于实验性阶段（Experimental）。未来版本中，指标名称可能调整，并将增加更多详细数据。

![](./images/ShaderStats.png)

| 属性              | 描述                                         |
| ----------------- | -------------------------------------------- |
| ConstantBuffer Count | 显示 Shader 使用的 ConstantBuffer 数量 |
| NumSamplers | 显示 Shader 使用的 Sampler 数量 |
| Resources Count | 显示 Shader 使用的资源数量 |
| Attributes Used | 显示 Shader 使用的顶点属性 |
| Texture Sample Count | 显示 Shader 贴图采样的次数 |
| Total Samplers | 显示 Shader 总共的 Sampler 数量 |
