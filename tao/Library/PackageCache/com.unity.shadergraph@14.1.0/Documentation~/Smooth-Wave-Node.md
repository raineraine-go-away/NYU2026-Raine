SmoothWave 节点
================  

描述  
----  
生成基于输入 **UV** 的平滑波形纹理图案（SmoothWave）。通过调整 **Tiling** 参数控制波形的平铺密度，**Wavelength** 参数控制波的长度，**Amplitude** 参数调整波形的高度振幅，**Width** 参数控制波形的线宽。控件 **Anti Aliasing** 提供抗锯齿选项以优化图案的视觉效果。

端口  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制波形纹理的平铺密度（X 和 Y）       |  
| Wavelength   | 输入  | Float    | 无   | 控制波形的长度                         |  
| Amplitude    | 输入  | Float    | 无   | 控制波形的高度振幅                     |  
| Width        | 输入  | Float    | 无   | 控制波形的线宽                         |  
| Out          | 输出  | Float    | 无   | 输出值                       |  

**控件**  
------  
| 名称         | 类型       | 选项                              | 描述                                    |  
|--------------|------------|-----------------------------------|-----------------------------------------|  
| Anti Aliasing| 下拉菜单   | Fastest、Smooth、Derivative           | 调整抗锯齿设置以优化波形纹理的平滑效果 |
