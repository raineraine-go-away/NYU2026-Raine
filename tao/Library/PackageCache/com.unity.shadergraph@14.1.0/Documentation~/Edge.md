边
=


描述
--


**边**（Edge）定义了两个[端口](Port.md)（Ports）之间的连接。**边**决定了数据如何流经 Shader Graph 节点网络。边只能从输入端口连接到输出端口。


每条**边**都有一个[数据类型](Data-Types.md)用于定义可以连接到的端口。每种数据类型都有一个相关的颜色用于标识其类型。

可以使用鼠标左键单击端口并拖动来创建新**边**。可以使用 Delete (Windows) 和 Command \+ Backspace (OSX) 键删除边，或者通过右键单击[节点](Node.md)从上下文菜单中执行此操作。


通过使用鼠标左键从端口拖动**边**并将其放到工作空间的空白区域中，即可打开 [Create Node Menu](Create-Node-Menu.md) 上下文菜单。
