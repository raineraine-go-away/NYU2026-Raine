GammaToLinearSpaceExact 节点  
================  

**描述**  
----  
该节点用于将经过伽马校正（非线性）的数值转换为线性空间中的数值，输出更接近物理实际的值。伽马校正通常用于处理图像或纹理数据的非线性特性，使用此节点可以恢复其真实的物理属性，确保渲染过程中更准确的光照计算。


**端口**  
------  
| 名称         | 方向  | 类型       | 绑定          | 描述                                          |  
|--------------|-------|------------|---------------|-----------------------------------------------|  
| In           | 输入  | Float      | 无            | 输入经过伽马校正的数值              |  
| Out          | 输出  | Float      | 无            | 输出转换后的线性数值       |
