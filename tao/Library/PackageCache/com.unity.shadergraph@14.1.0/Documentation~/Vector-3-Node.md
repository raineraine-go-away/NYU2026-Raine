
Vector 3 节点
===========


描述
--


在着色器中定义 **Vector 3** 值。如果未使用[边](Edge.md)连接[端口](Port.md) **X**、**Y** 和 **Z**，此[节点](Node.md)定义一个常量 **Vector 3**，否则此[节点](Node.md)可用于组合各种 **Float** 值。


端口
--




| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| X | 输入 | Float | 无 | 输入 x 分量值 |
| Y | 输入 | Float | 无 | 输入 y 分量值 |
| Z | 输入 | Float | 无 | 输入 z 分量值 |
| Out | 输出 |  Vector 3 | 无 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
float3 _Vector3_Out = float3(X, Y, Z);

```
