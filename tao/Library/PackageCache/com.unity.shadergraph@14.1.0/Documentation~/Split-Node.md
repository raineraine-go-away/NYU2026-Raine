
Split 节点
========


描述
--

将输入矢量 **In** 分成四个 **Float** 输出：**R**、**G**、**B** 和 **A**。这些输出矢量由输入 **In** 的各个通道定义：分别是红色、绿色、蓝色和 Alpha。如果输入矢量 **In** 的维度小于 4（**Vector 4**），则输入中不存在的输出值将为 0。


端口
--
| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| In | 输入 | 动态矢量 | 无 | 输入值 |
| R | 输出 | Float | 无 | 输入的红色通道 |
| G | 输出 | Float | 无 | 输入的绿色通道 |
| B | 输出 | Float | 无 | 输入的蓝色通道 |
| A | 输出 | Float | 无 | 输入的 Alpha 通道 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
float _Split_R = In[0];
float _Split_G = In[1];
float _Split_B = 0;
float _Split_A = 0;

```
