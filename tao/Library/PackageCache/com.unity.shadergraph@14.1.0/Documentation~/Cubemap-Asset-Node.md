
Cubemap Asset 节点
================


描述
--


定义要在着色器中使用的常量**立方体贴图资源**（Cubemap Asset）。要对**立方体贴图资源**采样，必须将其与 [Sample Cubemap 节点](Sample-Cubemap-Node.md)结合使用。使用单个 **Cubemap Asset 节点**时，可使用不同的参数对**立方体贴图**进行两次采样，无需对**立方体贴图**本身进行两次定义。


端口
--

| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| Out | 输出 | 立方体贴图 | 无 | 输出值 |


控件
--

| 名称 | 类型 | 选项 | 描述 |
| --- | --- | --- | --- |
|  | 对象字段（立方体贴图） |  | 定义项目中的立方体贴图资源。 |
