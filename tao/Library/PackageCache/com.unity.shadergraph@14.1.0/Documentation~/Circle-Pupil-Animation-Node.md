
Circle Pupil Animation 节点
=========================

描述
---

该节点将变形应用于归一化的 IrisUV 坐标，以模拟瞳孔的打开和关闭。


渲染管线兼容性
-------

| **节点** | **通用渲染管线 (URP)** | **高清渲染管线 (HDRP)** |
| --- | --- | --- |
| **Circle Pupil Animation 节点** | 否 | 是 |


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| **IrisUV** | 输入 | Vector2 | 对象空间中要着色的片元的位置。 |
| **Pupil Radius** | 输入 | Float | 对象空间中的入射线方向。或者来自光栅化中的摄像机，或者来自光线追踪中的上一反弹。 |
| **Maximal Pupil Aperture** | 输入 | Float | 对象空间中眼睛表面的法线。 |
| **Minimal Pupil Aperture** | 输入 | Float | 眼睛的折射率（默认为 **1.333**）。 |
| **Pupil Apertur** | 输入 | Float | 角膜末端与虹膜平面之间的距离。对于默认模型，此值应为 **0\.02** |
| **IrisUV** | 输出 | Vector2 | 对象空间中，折射点在虹膜平面上的位置。 |
