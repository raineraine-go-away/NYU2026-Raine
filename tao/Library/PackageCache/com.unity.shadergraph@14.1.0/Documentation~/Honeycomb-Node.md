Honeycomb 节点
================  

描述
----  
生成基于输入 **UV** 的蜂窝状纹理图案（Honeycomb）。通过调整 **Tiling** 参数控制蜂窝的平铺密度，**Scale** 参数控制蜂窝单元的整体缩放比例，**Edge Width** 参数用于调整蜂窝边框的宽度。

端口 
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制蜂窝纹理的平铺密度（X 和 Y）       |  
| Scale        | 输入  | Float    | 无   | 控制蜂窝单元的整体缩放比例             |  
| Edge Width   | 输入  | Float    | 无   | 控制蜂窝单元边框的宽度                 |  
| Out          | 输出  | Float    | 无   | 输出值                       |
