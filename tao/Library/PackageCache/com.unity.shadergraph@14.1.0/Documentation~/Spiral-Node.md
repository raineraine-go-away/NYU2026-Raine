Spiral 节点 
================  

描述 
----  
生成基于输入 **UV** 的螺旋状纹理图案（Spiral）。通过调整 **Tiling** 参数控制螺旋的平铺密度，**Position** 参数设置螺旋中心的位置，**Number** 参数控制螺旋的线条数，**Width** 参数调节螺旋线的宽度，**Separation** 参数用于调整螺旋线间的距离。控件 **Anti Aliasing** 提供抗锯齿选项以优化图案的视觉效果。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制螺旋纹理的平铺密度（X 和 Y）       |  
| Position     | 输入  | Vector 2 | 无   | 设置螺旋中心的位置                     |  
| Number       | 输入  | Float    | 无   | 控制螺旋的线条数                         |  
| Width        | 输入  | Float    | 无   | 调节螺旋线的宽度                       |  
| Separation   | 输入  | Float    | 无   | 调整螺旋线之间的距离                   |  
| Out          | 输出  | Float    | 无   | 输出值                       |  

**控件**  
------  
| 名称         | 类型       | 选项                              | 描述                                    |  
|--------------|------------|-----------------------------------|-----------------------------------------|  
| Anti Aliasing| 下拉菜单   | Fastest、Smooth、Derivative       | 调整抗锯齿设置以优化螺旋纹理的平滑效果 |
