材质变体
=================

在项目中创建材质时，您可能希望基于单一材质创建变体：例如，具有不同颜色方案的服装、受损和未受损的场景版本，或者光亮和风化的道具实例。

您可以使用材质变体来管理这些变体。有关材质变体的更多信息，请参见团结引擎用户手册中的 [材质变体](https://docs.unity.cn/cn/tuanjiemanual/Manual/materialvariant-landingpage.html)。

您可以从任何 Shader Graph 材质创建材质变体。有关如何创建材质变体的更多信息，请参见用户手册中的 [创建、修改和应用材质变体](https://docs.unity.cn/cn/tuanjiemanual/Manual/materialvariant-tasks.html)。
