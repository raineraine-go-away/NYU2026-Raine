
Divide 节点
=========


描述
--


返回输入 **A**（被除数）除以输入 **B**（除数）的结果。


端口
--




| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| A | 输入 | 动态矢量 | 第一个输入值 |
| B | 输入 | 动态矢量 | 第二个输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Divide_float4(float4 A, float4 B, out float4 Out)
{
    Out = A / B;
}

```
