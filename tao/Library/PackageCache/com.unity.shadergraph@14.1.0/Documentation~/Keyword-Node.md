
Keyword 节点
==========

描述
--

您可以使用 Keyword 节点在 Shader Graph 中创建一个静态分支，该分支引用一个位于 [Blackboard](Blackboard.md) 上的[关键字](Keywords.md)。Keyword 节点的外观，包括其可用端口，会根据它引用的关键字而变化。


创建新的 Keyword 节点
---------------

因为每个 Keyword 节点引用一个特定的关键字，所以必须首先在 Blackboard 上定义至少一个关键字。将一个关键字从 Blackboard 拖到工作区，以创建与该关键字对应的 Keyword 节点。


还可以右键单击工作区的任意位置，然后使用 **Create Node** 菜单以创建新的 Keyword 节点。**Keywords** 下方列出可在 Blackboard 上定义的关键字。单击该列表中的一个关键字即可创建相应的 Keyword 节点。
