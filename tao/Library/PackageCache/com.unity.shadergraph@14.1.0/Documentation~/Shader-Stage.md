
着色器阶段
=====

描述
--

**着色器阶段（Shader Stage）** 是指[节点](Node.md)或[端口](Port.md)所属的着色器管线的一部分。例如，**顶点**或**片元**。


在 Shader Graph 中，**着色器阶段**是针对每个端口定义的，但一个节点上的所有端口通常都锁定到同一个**着色器阶段**。由于底层着色器语言的限制，有些节点上的端口在某些**着色器阶段**中不可用。请参阅[节点库](Node-Library.md)文档了解存在**着色器阶段**限制的节点。


着色器阶段列表
-------

| 名称 | 描述 |
| --- | --- |
| Vertex | 按每个顶点计算的运算 |
| Fragment | 按每个片元计算的运算 |
