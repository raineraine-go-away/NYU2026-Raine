
Iris Out of Bound Color Clamp 节点
================================

描述
---

将虹膜颜色限制为给定的颜色。这在折射光线到达角膜内部的情况下很有用。


渲染管线兼容性
-------

| **节点** | **通用渲染管线 (URP)** | **高清渲染管线 (HDRP)** |
| --- | --- | --- |
| **Iris Out of Bound Color Clamp 节点** | 否 | 是 |


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| **IrisUV** | 输入 | Vector2 | 标准化的 UV 坐标，用于对纹理进行采样或以程序方式生成虹膜纹理。 |
| **Iris Color** | 输入 | Color | 之前采样或生成的虹膜颜色。 |
| **Clamp Color** | 输入 | Color | 要将虹膜限制成的颜色。 |
| **Iris Color** | 输出 | Color | 为剩余管线生成的虹膜颜色。 |
