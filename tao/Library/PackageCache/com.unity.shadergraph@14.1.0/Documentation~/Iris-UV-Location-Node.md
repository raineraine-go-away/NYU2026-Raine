
Iris UV Location 节点
===================

描述
---

该节点将角膜/虹膜的对象位转换为 UV 采样坐标。


渲染管线兼容性
-------

| **节点** | **通用渲染管线 (URP)** | **高清渲染管线 (HDRP)** |
| --- | --- | --- |
| **Iris UV Location 节点** | 否 | 是 |


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| **Position OS** | 输入 | Vector3 | 对象空间中位于虹膜平面上的位置。 |
| **Iris Radius** | 输入 | Float | 虹膜在所用模型中的半径。对于默认模型，此值应为 **0.225**。 |
| **IrisUV** | 输出 | Vector2 | 标准化的 UV 坐标，可用于对纹理进行采样或以程序方式生成虹膜纹理。 |
