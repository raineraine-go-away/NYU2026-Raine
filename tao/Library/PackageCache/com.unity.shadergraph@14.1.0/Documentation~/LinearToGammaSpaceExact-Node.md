LinearToGammaSpaceExact 节点 
================  

**描述**  
----  
该节点将线性颜色空间中的数值精确转换为伽马颜色空间中的数值，适用于将计算或渲染结果转换为适合显示设备的格式。伽马校正是为了适应显示设备的非线性响应，通常用于图像渲染和显示处理。


**端口**  
------  
| 名称         | 方向  | 类型       | 绑定          | 描述                                          |  
|--------------|-------|------------|---------------|-----------------------------------------------|  
| In           | 输入  | Float      | 无            | 输入线性颜色空间中的数值                    |  
| Out          | 输出  | Float      | 无            | 输出转换后的伽马颜色空间中的数值              |
