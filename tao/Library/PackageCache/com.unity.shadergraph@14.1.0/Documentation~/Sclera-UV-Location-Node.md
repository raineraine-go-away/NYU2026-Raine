
Sclera UV Location 节点
=====================

描述
---

该节点将巩膜的对象位转换为 UV 采样坐标。


渲染管线兼容性
-------




| **节点** | **通用渲染管线 (URP)** | **高清渲染管线 (HDRP)** |
| --- | --- | --- |
| **Sclera UV Location 节点** | 否 | 是 |


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| **PositionOS** | 输入 | Vector3 | **对象空间中要着色的片元的位置。** |
| **ScleraUV** | 输出 | Vector2 | 标准化的 UV 坐标，可用于对纹理进行采样或以程序方式生成巩膜纹理。 |
