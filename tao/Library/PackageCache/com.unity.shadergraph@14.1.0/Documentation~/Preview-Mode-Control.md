预览模式控制
=================

[](#description)描述
-------------------

该控制项允许您手动选择具有预览功能的节点的预览模式。

当在**预览模式控制**（Preview mode control）中选择 **Inherit** 时，编辑器会自动选择要使用的预览模式。

该选择由您正在预览的节点类型、子图设置（如果该节点位于子图中）或其他上游节点决定。要覆盖继承模式，请选择 **Preview 2D** 或 **Preview 3D**。

该模式控制功能同样适用于子图的预览。参见 [图表设置选项卡](Graph-Settings-Tab.md)。

[](#how-to-use)使用方法
---------------------

对于节点：

1.  添加包含预览的节点。
2.  选择该节点。
3.  在图表检查器或节点设置中找到预览控制。
4.  选择一个选项。

对于子图：

*   在子图的图表设置菜单中选择模式。
