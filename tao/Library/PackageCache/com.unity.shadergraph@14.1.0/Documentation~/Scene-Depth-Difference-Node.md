Scene Depth Difference 节点
======================

[](#description)描述
---------------------------

提供一个世界空间位置（World Space Position）和指定 UV 的深度值之间的差异。


[](#ports)端口
---------------
| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| Scene UV | 输入 | Vector 4 | 无 | 用于采样深度的 UV。 |
| Position WS | 输入 | Vector 3 | 无 | 用于与场景深度比较的世界空间位置。 |
| Out | 输出 | Float | 无 | PositionWS 与深度的差值。差值的相对模式有：**Eye** 模式与相机对齐，**Raw** 模式为深度缓冲区值，**Linear01** 模式为 0 到 1 的线性重映射值。 |

[](#controls)控件
---------------------
| 名称 | 类型 | 选项 | 描述 |
| --- | --- | --- | --- |
| Mode | 下拉列表 | 选择 **Linear01** 获取 0 到 1 之间的值，**Eye** 获取与场景单位匹配的世界空间值，或选择 **Raw** 用于 `SceneDepthBuffer`。 |
