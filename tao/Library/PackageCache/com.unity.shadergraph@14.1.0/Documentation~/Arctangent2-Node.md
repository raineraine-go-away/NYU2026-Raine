
Arctangent2 节点
==============


描述
--


返回输入 **A** 和输入 **B** 的值的反正切值。输入值的符号（无论是正值还是负值）用于确定输出分量或通道在 \-Pi 到 Pi 的范围内是正还是负。


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| A | 输入 | 动态矢量 | 第一个输入值 |
| B | 输入 | 动态矢量 | 第二个输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Arctangent2_float4(float4 A, float4 B, out float4 Out)
{
    Out = atan2(A, B);
}

```
