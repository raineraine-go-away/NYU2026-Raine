RGB to Grayscale 节点  
================  

**描述**  
----  
该节点输入 RGB 信息，并将其转换为灰度信息输出。通过将颜色的红、绿、蓝通道的加权平均值转换为单一的灰度值，实现从彩色图像到灰度图像的转换。通常用于图像处理和特效中，将彩色图像转换为黑白图像以突出形状或结构。


**端口**  
------  
| 名称         | 方向  | 类型       | 绑定          | 描述                                          |  
|--------------|-------|------------|---------------|-----------------------------------------------|  
| In           | 输入  | Vector 3     | 无            | 输入的 RGB 颜色信息（包含 R、G、B 三个通道）   |  
| Out          | 输出  | Float      | 无            | 输出转换后的灰度值                            |
