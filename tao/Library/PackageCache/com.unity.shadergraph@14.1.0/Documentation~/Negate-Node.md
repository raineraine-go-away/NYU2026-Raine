
Negate 节点
=========


描述
--


返回输入 **In** 的翻转符号值。正值变为负值，而负值变为正值。


端口
--




| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| In | 输入 | 动态矢量 | 输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Negate_float4(float4 In, out float4 Out)
{
    Out = -1 * In;
}

```
