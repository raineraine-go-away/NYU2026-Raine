
Screen 节点
=========


描述
--


允许访问屏幕的参数。


#### 支持的渲染管线


* 通用渲染管线（URP）


高清渲染管线（HDRP）**不**支持此节点。

注意：启用动态分辨率后，该节点将返回渲染摄像机的当前视口。经过升频后，此节点的输出将等于屏幕尺寸。



端口
--

| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| Width | 输出 | Float | 无 | 屏幕的宽度（以像素为单位） |
| Height | 输出 | Float | 无 | 屏幕的高度（以像素为单位） |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
float _Screen_Width = _ScreenParams.x;
float _Screen_Height = _ScreenParams.y;

```
