
Projection 节点
=============


描述
--


返回将输入 **A** 的值投影到与输入 **B** 的值平行的直线上的结果。


端口
--




| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| A | 输入 | 动态矢量 | 第一个输入值 |
| B | 输入 | 动态矢量 | 第二个输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Projection_float4(float4 A, float4 B, out float4 Out)
{
    Out = B * dot(A, B) / dot(B, B);
}

```
