Depth Fade 节点
===

描述
---

**Depth Fade** 节点输出一个线性渐变，表示物体表面与后方物体之间的距离，通过调节 **Distance** 参数来设置渐变的范围。该节点使用深度缓冲，因此需使用在透明物体中。


端口
---

| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| Distance | 输入 | Float | 无 | 输入的深度信息，决定渐变效果的范围 |
| Out | 输出 | Float | 无 | 输出值 |

控件
---
| 名称 | 类型 | 选项 | 描述 |
| --- | --- | --- | --- |
| Convert To Linear | 开关 | 无 | 将输出的渐变转换为线性 |
| Mirror | 开关 | 无 | 输出绝对值 |
| Saturate | 开关 | 无 | 输出值限制在（0，1） |
