Taiji 节点
================  

**描述**  
----  
生成基于输入 **UV** 的太极图案（Taiji）。通过调整 **Rotate** 参数控制太极图案的旋转角度，**Eye Size** 参数调节太极图中点的大小。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Rotate       | 输入  | Float    | 无   | 控制太极图案的旋转角度（单位：度）     |  
| Eye Size     | 输入  | Float    | 无   | 调节太极图中点的大小                   |  
| Out          | 输出  | Float    | 无   | 输出太极图案的主图纹理值               |  
| Alpha        | 输出  | Float    | 无   | 输出太极图案的透明度值                 |
