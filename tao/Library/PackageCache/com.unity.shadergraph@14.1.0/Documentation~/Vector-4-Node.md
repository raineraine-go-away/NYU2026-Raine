
Vector 4 节点
===========


描述
--


在着色器中定义 **Vector 4** 值。如果未使用[边](Edge.md)连接[端口](Port.md) **X**、**Y**、**Z** 和 **W**，此[节点](Node.md)定义一个常量 **Vector 4**，否则此[节点](Node.md)可用于组合各种 **Float** 值。


端口
--




| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| X | 输入 | Float | 无 | 输入 x 分量值 |
| Y | 输入 | Float | 无 | 输入 y 分量值 |
| Z | 输入 | Float | 无 | 输入 z 分量值 |
| W | 输入 | Float | 无 | 输入 w 分量值 |
| Out | 输出 |  Vector 4 | 无 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
float4 _Vector4_Out = float4(X, Y, Z, W);

```
