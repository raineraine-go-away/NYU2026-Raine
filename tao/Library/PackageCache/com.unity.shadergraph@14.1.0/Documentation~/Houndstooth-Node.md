Houndstooth 节点
================  

描述 
----  
生成基于输入 **UV** 的千鸟格纹理图案（Houndstooth）。通过调整 **Tiling** 参数控制图案的平铺密度，**Teeth** 参数用于控制千鸟格图案单元的形状复杂度。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制千鸟格纹理的平铺密度（X 和 Y）     |  
| Teeth        | 输入  | Float    | 无   | 控制千鸟格单元的形状复杂度             |  
| Out          | 输出  | Float    | 无   | 输出值                     |
