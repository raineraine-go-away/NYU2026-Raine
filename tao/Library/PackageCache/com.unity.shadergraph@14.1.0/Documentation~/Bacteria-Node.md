Bacteria 节点  
================  

描述  
----  
生成基于输入 **UV** 的细菌状纹理图案（Bacteria）。通过调整 **Tiling** 参数控制图案的平铺密度，通过 **Seed** 参数生成不同的随机图案变化。

端口  
------  
| 名称  | 方向  | 类型     | 绑定 | 描述                            |  
|-------|-------|----------|------|---------------------------------|  
| UV    | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射   |  
| Tiling| 输入  | Vector 2 | 无   | 控制纹理的平铺密度（X 和 Y）   |  
| Seed  | 输入  | Float    | 无   | 控制生成图案的随机种子         |  
| Out   | 输出  | Float    | 无   | 输出值  |
