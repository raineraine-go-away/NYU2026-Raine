
Step 节点
=======


描述
--


对于每个分量，如果输入 **In** 的值大于或等于输入 **Edge** 的值，则返回 1，否则返回 0。


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| Edge | 输入 | 动态矢量 | 步长值 |
| In | 输入 | 动态矢量 | 输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Step_float4(float4 Edge, float4 In, out float4 Out)
{
    Out = step(Edge, In);
}

```
