Whirl 节点 
================  

**描述**  
----  
生成基于输入 **UV** 的旋涡状纹理图案（Whirl）。通过调整 **Tiling** 参数控制图案的平铺密度，**Position** 参数设置旋涡中心的位置，**Count** 参数控制旋涡的叶片数量，**Width** 参数调节叶片的宽度，**Rotation** 参数设置旋涡的整体旋转角度。控件 **Anti Aliasing** 提供抗锯齿选项以优化视觉效果。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制旋涡纹理的平铺密度（X 和 Y）       |  
| Position     | 输入  | Vector 2 | 无   | 设置旋涡中心的位置                     |  
| Count        | 输入  | Float    | 无   | 控制旋涡叶片的数量                     |  
| Width        | 输入  | Float    | 无   | 调节旋涡叶片的宽度                     |  
| Rotation     | 输入  | Float    | 无   | 设置旋涡的整体旋转角度（单位：度）     |  
| Out          | 输出  | Float    | 无   | 输出值                       |  

**控件**  
------  
| 名称         | 类型       | 选项                              | 描述                                    |  
|--------------|------------|-----------------------------------|-----------------------------------------|  
| Anti Aliasing| 下拉菜单   | Fastest、Smooth、Derivative       | 调整抗锯齿设置以优化旋涡纹理的平滑效果 |
