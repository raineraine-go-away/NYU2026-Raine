
Billboard 节点  
================  

**描述**  
----  
通过旋转顶点数据（位置、法线、切线），使物体的坐标轴与当前相机对齐。根据 **Billboard Mode** 的设置，可以选择对齐所有轴向（All Axis）或仅对齐 X 和 Z 轴（Around Y Axis）。此节点的输出适用于顶点数据，通常用于实现面向相机的效果（例如粒子系统中的板贴效果）。

**注意事项**  
- Billboard 节点仅适用于连接到顶点的端口（Vertex Ports），无法应用于片段数据。  


**端口**  
------  
| 名称           | 方向  | 类型       | 绑定         | 描述                                      |  
|----------------|-------|------------|--------------|-------------------------------------------|  
| Position OS    | 输入  | Vector 3   | Object Space | 输入的物体空间顶点位置                    |  
| Normal OS      | 输入  | Vector 3   | Object Space | 输入的物体空间法线                        |  
| Tangent OS     | 输入  | Vector 3   | Object Space | 输入的物体空间切线                        |  
| Position       | 输出  | Vector 3   | 无           | 输出旋转后的物体空间顶点位置              |  
| Normal         | 输出  | Vector 3   | 无           | 输出旋转后的物体空间法线                  |  
| Tangent        | 输出  | Vector 3   | 无           | 输出旋转后的物体空间切线                  |  

**控件**  
------  
| 名称           | 类型       | 选项                              | 描述                                      |  
|----------------|------------|-----------------------------------|-------------------------------------------|  
| Billboard Mode | 下拉菜单   | All Axis、Around Y Axis           | 设置对齐模式：对齐所有轴向或仅对齐 X 和 Z 轴 |
