
DDY 节点
======


描述
--


返回输入 **In** 相对于屏幕空间 Y 坐标的偏导数。此节点只能在像素着色器阶段中使用。


端口
--




| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| In | 输入 | 动态矢量 | 输入值 |
| Out | 输出 | 动态矢量 | 输出偏导数值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_DDY_float4(float4 In, out float4 Out)
{
    Out = ddy(In);
}

```
