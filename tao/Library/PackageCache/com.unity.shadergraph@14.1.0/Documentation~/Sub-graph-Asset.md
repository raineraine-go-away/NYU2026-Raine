
子图资源
=====

描述
--

**子图资源**是随 Shader Graph 引入的新**资源**类型。**子图资源**用于定义[子图](Sub-graph.md)。这不同于 Shader Graph。可使用 [Project 窗口](https://docs.unity.cn/cn/tuanjiemanual/Manual/ProjectView.html)中的 **Create** 菜单，通过 **Shader** 子菜单中的 **Sub Graph** 创建**子图资源**。


可通过双击**子图资源**或在选中**子图资源**时单击 [Inspector](https://docs.unity.cn/cn/tuanjiemanual/Manual/UsingTheInspector.html) 中的 **Open Shader Editor** 来打开 [Shader Graph 窗口](Shader-Graph-Window.md)。
