ZigZag 节点  
================  

**描述**  
----  
生成基于输入 **UV** 的锯齿状纹理图案（ZigZag）。通过调整 **Tiling** 参数控制图案的平铺密度，**Width** 参数设置锯齿线的宽度，**Wavelength** 参数控制锯齿的波长，**Amplitude** 参数用于调整锯齿的振幅。控件 **Anti Aliasing** 提供抗锯齿选项以优化图案的平滑效果。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制锯齿纹理的平铺密度（X 和 Y）       |  
| Width        | 输入  | Float    | 无   | 设置锯齿线的宽度                       |  
| Wavelength   | 输入  | Float    | 无   | 控制锯齿的波长                         |  
| Amplitude    | 输入  | Float    | 无   | 调整锯齿的振幅                         |  
| Out          | 输出  | Float    | 无   | 输出值                       |  

**控件**  
------  
| 名称         | 类型       | 选项                              | 描述                                    |  
|--------------|------------|-----------------------------------|-----------------------------------------|  
| Anti Aliasing| 下拉菜单   | Fastest、Smooth、Derivative       | 调整抗锯齿设置以优化锯齿纹理的平滑效果 |
