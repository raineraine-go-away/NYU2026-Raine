Shader Graph 快捷键参考
=========================================

Shader Graph 提供了多种键盘快捷键，帮助你更高效地工作。本页列出了 Shader Graph 中可用的快捷键。你可以使用这些快捷键来执行许多任务，例如：

*   向图中添加节点。
*   对节点进行分组和取消分组。
*   打开节点文档。
*   切换某些 UI 元素的显示和隐藏。

你还可以使用 快捷键管理器（Shortcuts Manager） 根据自己的喜好自定义键盘快捷键。有关如何自定义的更多信息，请参阅 [快捷键管理器](https://docs.unity.cn/cn/tuanjiemanual/Manual/ShortcutsManager.html)。

[](#built-in-keyboard-shortcuts)内置快捷键
-----------------------------------------------------------

下表列出了 Shader Graph 中可用的内置快捷键。你无法再次自定义这些内置快捷键。

| Command | Windows | macOS | Description |
| --- | --- | --- | --- |
| Frame All | A | A | 在 [Shader Graph 窗口](Shader-Graph-Window.md) 中显示整个图。 |
| Frame Selection | F | F | 在 [Shader Graph 窗口](Shader-Graph-Window.md) 中显示选中的节点。 |
| Duplicate | Ctrl + D | Cmd + D | 复制选中的节点。 |
| Copy | Ctrl + C | Cmd + C | 将选中的节点复制到剪贴板。 |
| Cut | Ctrl + X | Cmd + X | 从图中移除选中的节点并将其放入剪贴板。 |
| Paste | Ctrl + V | Cmd + V | 从剪贴板粘贴节点。 |
| Undo | Ctrl + Z | Cmd + Z | 撤销上一个操作。 |
| Redo | Ctrl + Y | Cmd + Y | 重做上一个操作。 |
| Open Create Node menu | Space | Space | 打开 [Create Node Menu](Create-Node-Menu.md) 以向图中添加节点。 |

[](#customizable-shortcuts)可自定义快捷键
-------------------------------------------------

下表列出了可通过 [快捷键管理器](https://docs.unity.cn/cn/tuanjiemanual/Manual/ShortcutsManager.html) 自定义的 Shader Graph 快捷键。

| Command | Windows | macOS | Description |
| --- | --- | --- | --- |
| Close Tab | Ctrl + F4 | Cmd + F4 | 关闭选中的标签页。 |
| Cycle Color Mode | Shift + 4 | Shift + 4 | 循环切换不同的颜色模式。 |
| Group Selected Nodes | Ctrl + G | Cmd + G | 将选中的节点进行分组。 |
| Insert Redirect Node | Ctrl + R | Cmd + R | 在选中的连接中插入一个重定向节点。 |
| Open Documentation | F1 | F1 | 打开选中节点的文档。 |
| Save | Ctrl + S | Cmd + S | 保存你的图。 |
| Save As... | Ctrl + Shift + S | Cmd + Shift + S | 以新名称保存你的图。 |
| Toggle Blackboard | Shift + 1 | Shift + 1 | 切换 [Blackboard](Blackboard.md) 的显示与隐藏。 |
| Toggle Inspector | Shift + 2 | Shift + 2 | 切换 [Graph Inspector](Internal-Inspector.md) 的显示与隐藏。 |
| Toggle Main Preview | Shift + 3 | Shift + 3 | 切换 [Main Preview](Main-Preview.md) 的显示与隐藏。 |
| Toggle Node Collapsed | Ctrl + P | Cmd + P | 切换选中节点的折叠状态。 |
| Toggle Node Previews | Ctrl + T | Cmd + T | 切换选中节点的预览显示状态。 |
| Ungroup Nodes | Ctrl + U | Cmd + U | 取消选中节点的分组。 |
