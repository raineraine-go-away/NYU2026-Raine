RGB to Luminance 节点  
================  

**描述**  
----  
该节点输入 RGB 信息，并将其转换为流明度信息输出。流明度代表图像亮度的感知值，通常通过对 RGB 通道进行加权平均计算得到。该节点可以用于从颜色数据中提取亮度信息，通常用于图像亮度调整、光照计算等场景。


**端口**  
------  
| 名称         | 方向  | 类型       | 绑定          | 描述                                          |  
|--------------|-------|------------|---------------|-----------------------------------------------|  
| In           | 输入  | Vector 3     | 无            | 输入的 RGB 颜色信息（包含 R、G、B 三个通道）   |  
| Out          | 输出  | Float      | 无            | 输出计算后的流明度值                          |
