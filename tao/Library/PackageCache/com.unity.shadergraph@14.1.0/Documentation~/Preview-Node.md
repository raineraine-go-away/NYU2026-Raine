
Preview 节点
==========


描述
--

提供预览窗口并传递输入值而无需修改。如果用户通常更喜欢折叠[节点](Node.md)预览，则可以使用此[节点](Node.md)在 Shader Graph 中的特定点提供预览。


端口
--

| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| In | 输入 | 动态矢量 | 无 | 输入值 |
| Out | 输出 | 动态矢量 | 无 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Preview_float4(float4 In, out float4 Out)
{
    Out = In;
}

```
