
Arcsine 节点
==========


描述
--


返回输入 **In** 的每个分量的反正弦值，作为相同维度和相等长度的矢量。每个分量都应在 \-Pi/2 到 Pi/2 的范围内。


端口
--




| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| In | 输入 | 动态矢量 | 输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Arcsine_float4(float4 In, out float4 Out)
{
    Out = asin(In);
}

```
