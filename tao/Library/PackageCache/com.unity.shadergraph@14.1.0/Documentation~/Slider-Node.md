
Slider 节点
=========


描述
--


使用 **Slider** 字段在着色器中定义一个常量 **Float** 值。可在 **Mode** 设置为 **Slider** 的情况下通过[节点](Node.md)的上下文菜单转换为 **Float** 类型的[属性](Property-Types.md)。


端口
--




| 名称 | 方向 | 类型 | 绑定 | 描述 |
| --- | --- | --- | --- | --- |
| Out | 输出 | Float | 无 | 输出值 |


控件
--




| 名称 | 类型 | 选项 | 描述 |
| --- | --- | --- | --- |
|  | Slider |  | 定义输出值。 |
| Min | Float |  | 定义滑动条参数的最小值。 |
| Max | Float |  | 定义滑动条参数的最大值。 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
float _Slider_Out = 1.0;

```
