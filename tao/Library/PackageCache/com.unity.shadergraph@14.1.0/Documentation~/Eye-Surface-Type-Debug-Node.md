
Eye Surface Type Debug 节点
=========================

描述
---

可用于直观地验证当前瞳孔半径的 Debug 节点。


渲染管线兼容性
-------

| **节点** | **通用渲染管线 (URP)** | **高清渲染管线 (HDRP)** |
| --- | --- | --- |
| **Eye Surface Type Debug 节点** | 否 | 是 |


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| **PositionOS** | 输入 | Vector3 | 要着色的片元在对象空间中的位置。 |
| **EyeColor** | 输入 | Color | 眼睛的最终漫射颜色。 |
| **IrisRadius** | 输入 | Float | 虹膜在所用模型中的半径。对于默认模型，此值应为 **0.225**。 |
| **Pupil Radius** | 输入 | Float | 虹膜纹理中的瞳孔半径（采用百分比）。 |
| **IsActive** | 输入 | bool | 定义节点是否应处于活动状态的标志。 |
| **SurfaceColor** | 输出 | Color | 眼睛的最终漫射颜色。 |
