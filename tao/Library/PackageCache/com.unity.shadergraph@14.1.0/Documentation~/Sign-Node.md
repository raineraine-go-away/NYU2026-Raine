
Sign 节点
=======


描述
--


对于每个分量，如果输入 **In** 的值小于零，则返回 \-1，如果等于零，则返回 0，如果大于零，则返回 1。


端口
--




| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| In | 输入 | 动态矢量 | 输入值 |
| Out | 输出 | 动态矢量 | 输出值 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
void Unity_Sign_float4(float4 In, out float4 Out)
{
    Out = sign(In);
}

```
