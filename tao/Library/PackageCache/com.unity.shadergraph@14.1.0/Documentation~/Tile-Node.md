Tile 节点  
================  

**描述**  
----  
生成基于输入 **UV** 的地板状纹理图案（Tile）。通过调整 **Width** 和 **Height** 参数控制单个地板的宽度和高度，**Factor** 参数调整地板之间的偏移比例，**Alignment** 参数设置地板的对齐方式，**Stroke** 参数调节地板边框的宽度。控件提供平滑和锐利两种样式选择。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Width        | 输入  | Float    | 无   | 控制单个地板的宽度                     |  
| Height       | 输入  | Float    | 无   | 控制单个地板的高度                     |  
| Factor       | 输入  | Float    | 无   | 调整地板之间的偏移比例                 |  
| Alignment    | 输入  | Float    | 无   | 设置地板的对齐方式                     |  
| Stroke       | 输入  | Float    | 无   | 控制地板边框的宽度                     |  
| Out          | 输出  | Float    | 无   | 输出值                       |  

**控件**  
------  
| 名称         | 类型       | 选项                              | 描述                                    |  
|--------------|------------|-----------------------------------|-----------------------------------------|  
|        | 下拉菜单   | Smooth、Sharp                    | 设置地板的样式（平滑或锐利）       |
