PixelateUV 节点
================  

**描述**  
----  
该节点用于将输入的 UV 信息进行像素化处理，从而在纹理采样时产生马赛克效果。通过降低 UV 的分辨率，使得纹理的细节被压缩，从而达到视觉上像素化的效果。

**端口**  
------  
| 名称         | 方向  | 类型       | 绑定          | 描述                                          |  
|--------------|-------|------------|---------------|-----------------------------------------------|  
| UV           | 输入  | Vector 2    | 无            | 输入的 UV 坐标，用于纹理采样                  |  
| Pixels X    | 输入  | Float      | 无            | 控制像素化的X轴方向大小            |
| Pixels Y    | 输入  | Float      | 无            | 控制像素化的Y轴方向大小            |
| Out          | 输出  | Vector 2    | 无            | 输出经过像素化处理后的 UV 坐标               |
