
Texture 3D Asset 节点
===================


描述
--


定义要在着色器中使用的常量 **3D 纹理资源**。要对 **3D 纹理资源**采样，必须将其与 [Sample Texture 3D 节点](Sample-Texture-3D-Node.md)结合使用。使用单个 **Texture 3D Asset 节点**时，可使用不同的参数对 **3D 纹理**进行两次采样，无需对 **3D 纹理**本身进行两次定义。


端口
--

| 名称 | 方向 | 类型 | 描述 |
| --- | --- | --- | --- |
| Out | 输出 | 3D 纹理 | 输出值 |


控件
--

| 名称 | 类型 | 选项 | 描述 |
| --- | --- | --- | --- |
|  | 对象字段（3D 纹理） |  | 定义项目中的 3D 纹理资源。 |


生成的代码示例
-------


以下示例代码表示此节点的一种可能结果。



```
TEXTURE3D(_Texture3DAsset); 
SAMPLER(sampler_Texture3DAsset);

```
