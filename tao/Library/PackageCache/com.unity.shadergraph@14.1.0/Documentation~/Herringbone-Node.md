Herringbone 节点
================  

描述
----  
生成基于输入 **UV** 的人字形纹理图案（Herringbone）。通过调整 **Tiling** 参数控制图案的平铺密度，**Width** 参数控制砖块接缝的宽度，**Cells** 参数调整人字形图案的单元格数量。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Tiling       | 输入  | Vector 2 | 无   | 控制人字形纹理的平铺密度（X 和 Y）     |  
| Width        | 输入  | Float    | 无   | 控制砖块接缝的宽度                     |  
| Cells        | 输入  | Float    | 无   | 调整人字形图案的单元格数量             |  
| Out          | 输出  | Float    | 无   | 输出值                     |
