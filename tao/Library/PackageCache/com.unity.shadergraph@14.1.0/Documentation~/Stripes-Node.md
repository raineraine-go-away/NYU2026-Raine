Stripes 节点
================  

**描述**  
----  
生成基于输入 **UV** 的条纹状纹理图案（Stripes）。通过调整 **Frequency** 参数控制条纹的重复频率，**Offset** 参数设置条纹的偏移位置，**Thickness** 参数调节条纹的宽度，**Rotation** 参数用于旋转条纹图案以适应不同的设计需求。

**端口**  
------  
| 名称         | 方向  | 类型     | 绑定 | 描述                                    |  
|--------------|-------|----------|------|-----------------------------------------|  
| UV           | 输入  | Vector 2 | UV   | 输入的 UV 坐标，用于纹理映射           |  
| Frequency    | 输入  | Float    | 无   | 控制条纹纹理的重复频率                 |  
| Offset       | 输入  | Float    | 无   | 设置条纹的偏移位置                     |  
| Thickness    | 输入  | Float    | 无   | 调节条纹的宽度                         |  
| Rotation     | 输入  | Float    | 无   | 控制条纹图案的旋转角度（单位：度）     |  
| Out          | 输出  | Float    | 无   | 输出值                       |
