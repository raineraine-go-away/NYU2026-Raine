# 通用渲染管线概述
![URP 渲染效果](Images/AssetShots/Beauty/Overview.png)

**通用渲染管线（Universal Render Pipeline，URP）** 是 Tuanjie 提供的 预构建可编程渲染管线 (Scriptable Render Pipeline)。

URP 提供了对美术师友好的工作流程，可让您在移动平台、高端游戏主机和 PC 等各种平台上快速轻松地创建优化的图形。

## 系统要求
有关 URP 的系统要求和兼容性信息，请参考 [系统要求](requirements.md)。

## URP 最新更新
了解 URP 最新版本的更新内容，请参考 [URP 更新内容](whats-new/urp-whats-new.md)。

## 开始使用 URP
如果你想从零开始创建 URP 项目，或在现有项目中安装 URP，请参考 [URP 入门指南](InstallingAndConfiguringURP.md)。
