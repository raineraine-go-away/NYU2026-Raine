# 预构建效果（渲染器特性）

渲染器特性是一种资产，可以让你向 Universal Render Pipeline (URP) 渲染器添加内置效果，并配置其行为。

有关如何使用渲染器特性的示例，请参考 [URP 包示例中的渲染器特性](package-sample-urp-package-samples.md#renderer-features)。

| 页面 | 描述 |
| ---- | ---- |
| [如何向渲染器添加渲染器特性](urp-renderer-feature-how-to-add.md) | 向渲染器添加渲染器特性。 |
| [渲染物体渲染器特性](renderer-features/renderer-feature-render-objects-landing.md) | 在特定的层上、特定的时间绘制物体，并使用特定的覆盖。 |
| [Decal 渲染器特性](renderer-feature-decal-landing.md) | 将特定材质（贴花）投影到场景中的其他物体上。贴花会与场景的光照交互，并包裹在网格周围。 |
| [屏幕空间环境光遮蔽（SSAO）渲染器特性](post-processing-ssao.md) | 实时地加深褶皱、孔洞、交点和相互靠近的表面。 |
| [屏幕空间阴影渲染器特性](renderer-feature-screen-space-shadows.md) | 计算受主要方向光影响的、不透明物体的屏幕空间阴影，并将其绘制到场景中。 |
| [全屏通道渲染器特性](renderer-features/renderer-feature-full-screen-pass-landing.md) | 全屏通道渲染器特性的参考。 |
