# 光照模式

您在光照窗口中选择的[光照模式](https://docs.unity.cn/cn/tuanjiemanual/Manual/lighting-mode.html)决定了当前场景中所有混合光源的行为。

URP支持以下光照模式：

* [烘焙间接光照](https://docs.unity.cn/cn/tuanjiemanual/Manual/LightMode-Mixed-BakedIndirect.html)
* [减法光照](https://docs.unity.cn/cn/tuanjiemanual/Manual/LightMode-Mixed-Subtractive.html)
* [阴影蒙版光照](https://docs.unity.cn/cn/tuanjiemanual/Manual/LightMode-Mixed-Shadowmask.html)
