# 为封闭平台构建项目

如果您拥有封闭平台（Closed platforms）的开发许可，并需要遵守平台提供商的保密和法律协议，请前往相应的开发者论坛，获取特定于该平台的渲染管线包（Render Pipeline Package）链接。

## 平台包安装

封闭平台的包**不可**通过 **Package Registry** 或 **Package Manager** 直接获取。

**安装封闭平台包的方法：**

1. 从相关平台的开发者论坛下载该包。
2. 使用 **Package Manager** 本地安装该包。有关本地安装包的详细步骤，请参考 [安装本地包](https://docs.unity.cn/cn/tuanjiemanual/Manual/upm-ui-local.html)。
