# 如何向渲染器添加渲染器特性

要向渲染器添加渲染器特性：

1. 在 __Project__ 窗口中，选择一个渲染器。

    ![选择一个渲染器。](Images/add-renderer-feature/renderer-feature-select-renderer.png)

    Inspector 窗口显示渲染器属性。

    ![Inspector 窗口显示渲染器属性。](Images/add-renderer-feature/renderer-feature-inspector-no-rend-features.png)

2. 在 Inspector 窗口中，选择 __Add Renderer Feature__。在列表中，选择一个渲染器特性。

    ![选择 __Add Renderer Feature__，然后选择一个渲染器特性。](Images/add-renderer-feature/renderer-feature-select-renderer-feature.png)

    Unity 将选中的渲染器特性添加到渲染器。

    ![新渲染器特性已添加。](Images/add-renderer-feature/renderer-feature-created.png)

Unity 将渲染器特性显示为渲染器在项目窗口中的子项：

![渲染器特性作为渲染器的子项显示在项目窗口中](Images/add-renderer-feature/renderer-feature-project-window.png)
