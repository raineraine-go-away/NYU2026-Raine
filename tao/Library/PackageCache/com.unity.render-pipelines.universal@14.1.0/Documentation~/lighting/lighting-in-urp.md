# 通用渲染管线中的光照

所有 Unity 渲染管线都共享通用的光照功能，但每个渲染管线在一些重要方面有所不同。

**Universal Render Pipeline (URP)** 与 Unity 通用光照功能的不同之处包括：

* [Light 组件检查器](../light-component.md)，它显示一些特定于 URP 的控制选项。
* [Universal Additional Light Data](../universal-additional-light-data.md) 组件，它允许 Unity 存储与 URP 相关的光照数据。

要查看 Unity 内置渲染管线和 URP 之间的光照功能的完整对比，以及当前正在研究中的光照功能的最新列表，请查阅 [Render pipeline feature comparison](https://docs.unity.cn/cn/tuanjiemanual/Manual/render-pipelines-feature-comparison.html)。

有关 Unity 中光照的总体介绍以及常见光照工作流的示例，请参考 [Unity 手册中的光照部分](https://docs.unity.cn/cn/tuanjiemanual/Manual/LightingOverview.html)。
